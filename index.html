Linha neutra adimensional (verificação):
ξ = As·fyd / (αc·fcd·bw·d·λ)

Momento resistente:
Mk = (As·fyd/γf)·d·(1 − λ·ξ/2)

Conceito — Armadura Dupla (NBR 6118)

Quando ξ calculado excede 0,45 (limite de ductilidade), a seção simples não é suficiente. Adiciona-se armadura de compressão (As,sup) para absorver o momento excedente ΔMd, mantendo ξ = 0,45 na armadura de tração inferior (As,inf).

Passo 1: Calcular ξ. Se ξ > 0,45 → adotar ξ = 0,45 e calcular armadura dupla
Passo 2: Md1 = αc·fcd·bw·d²·λ·ξ·(1−λξ/2)  →  As1 = Md1 / (fyd·d·(1−λξ/2))
Passo 3: ΔMd = Mdi − Md1  →  As2 = ΔMd / (fyd·(d−d''))
Resultado: As,inf = As1 + As2  |  As,sup = As2
